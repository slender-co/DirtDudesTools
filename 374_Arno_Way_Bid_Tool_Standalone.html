Bid Tool Guide & Glossary

This reference covers every abbreviation, column, and concept used throughout the tool. Intended for anyone reviewing or editing the bid who wasn't part of the original build process.

Controls Bar — Top of Page

Wall Length (LF) — Total linear footage of the retaining wall being bid. LF = Linear Feet. This drives the quantity for all line items set to "LF" qty mode.
Wall Height (FT) — Height of the retaining wall in feet. Used to calculate PSF (Price per Square Foot) metrics. FT = Feet.
Actual PLF — Your real cost per linear foot based on the total bid ÷ wall length. This updates live as you edit any pricing. PLF = Price per Linear Foot ($/LF).
Actual PSF — Your real cost per square foot based on total bid ÷ (wall length × wall height). PSF = Price per Square Foot ($/SF).
ROM Target — Rough Order of Magnitude. The small editable number beneath each badge. This is your planning estimate to compare against — not a calculation input. The delta text ("+$X over" or "-$X under") shows how your actual bid compares to this target.
Base Bid Total — Sum of all line items across all sections: Material + Labor + Equipment for every row.

Bid Table — Column Definitions

# — Sequential row number. Auto-assigned.
Description — Text field describing the scope item. Fully editable.
Qty — Quantity for this line item. Click to cycle between three modes: LF (auto-fills from Wall Length), LS (Lump Sum = 1), or EA (Each — manual entry for any custom quantity).
Unit — Unit of measurement. Displays LF, LS, or EA depending on Qty mode. Click to cycle along with Qty.
Unit Cost — Material unit cost in dollars. For LF items, this is the cost per linear foot of material only (no labor or equipment). Material $ = Qty × Unit Cost.
Material $ — Calculated: Qty × Unit Cost. Represents raw material cost for this line item. Highlighted in blue header columns.
Days — Duration in working days for this task. Supports fractional values (e.g. 0.5 = half day, 1.5 = day and a half). Purple-tinted input. Used to calculate auto labor/equipment costs when a Resource is assigned.
Resource — Dropdown populated from the Rates tab. Select a labor role or equipment type to auto-calculate cost. Set to "— Manual —" to type Labor $ or Equip $ directly. When a resource is assigned, the corresponding $ field shows "auto" tag and is computed as: Rate × Days × Count.
# (Count) — Number of workers or equipment units for this task. E.g., 2 iron workers × 3 days × $500/day = $3,000. Purple-tinted input.
Labor $ — Labor cost for this line item. Either auto-calculated (Rate × Days × Count when a labor resource is assigned) or manually entered. Green-tinted input. Labeled with green header.
Equip $ — Equipment cost for this line item. Same logic as Labor $ but for equipment resources. Amber-tinted input. Labeled with amber/brown header.
Line Total — Material $ + Labor $ + Equip $ for this row. This is the all-in cost for this scope item.
Notes — Free-form text for specifications, references, or context. Fully editable.

Qty Mode Reference

LF (Linear Foot) — Quantity auto-populates from the Wall Length control. Used for items that scale with wall length (concrete, rebar, pipe, etc.).
LS (Lump Sum) — Quantity = 1. Used for fixed-cost items (mobilization, permits, engineering, crew labor days, etc.).
EA (Each) — Manual quantity entry. Used for items with custom counts (cubic yards of import fill, number of inspections, etc.).

Rates Tab

Rate Card — A named resource with a daily rate. Two categories: Labor (green) and Equipment (amber). These populate the Resource dropdown on every bid row.
Day Rate — The cost per day for this resource. Fractional days are supported: 0.8 days × $250/day = $200.00.
Auto-calc Formula — When a resource is assigned to a bid row: Rate × Days × Count = Labor $ or Equip $. The result appears with a purple "auto" tag.

PLF Breakdown Tab

PLF (Price per Linear Foot) — Total cost per running foot of wall. Industry-standard metric for retaining wall pricing. Your $400 ROM = $300 wall stem + $100 footing.
Wall Stem — The vertical portion of the retaining wall above the footing. Includes concrete, rebar, formwork, and labor for the stem itself.
Footing — The horizontal concrete base that the wall sits on. Transfers wall loads into the soil. Design dimensions TBD by structural engineer.
Stacked Bar — Visual breakdown of how the PLF is distributed across cost components. Each segment's percentage is editable — adjust to see how cost allocation shifts.
Balance Indicator — Percentages must total 100%. Green = balanced. Red = needs adjustment.

Summary Tab

Material / Labor / Equipment Split — Stacked bar showing percentage of total bid in each cost category. Helps evaluate if your bid is material-heavy or labor-heavy.
Est. Duration — Sum of the peak (maximum) duration from each section. Gives a rough project timeline estimate.
ROM Baseline — Wall Length × ROM PLF Target. The "should-cost" number you're comparing your actual bid against.
Variance — Actual bid total minus ROM baseline. Positive = over budget. Negative = under budget. Card turns amber when over.

Abbreviations Quick Reference

CIP — Cast-In-Place. Concrete poured on-site into forms (vs. precast or CMU block).
CMU — Concrete Masonry Unit. Block wall construction (NOT used on this project — wall is CIP).
PLF — Price per Linear Foot ($/LF). Total cost divided by wall length.
PSF — Price per Square Foot ($/SF). Total cost divided by wall face area (length × height).
ROM — Rough Order of Magnitude. A planning-level cost estimate, not a firm quote.
LF — Linear Feet. Horizontal measurement of the wall.
LS — Lump Sum. A fixed total price regardless of quantity.
EA — Each. A per-unit count for items not measured in linear feet.
CY — Cubic Yard. Volume measure (27 cubic feet). Used for soil, concrete, gravel.
SF — Square Foot/Feet. Area measure (length × height).
SE — Structural Engineer. Licensed professional who designs the wall and footing.
CBC — California Building Code (2025 edition governs this project).
ROW — Right of Way. Public roadway or access area.
GL — General Liability (insurance).
WC — Workers' Compensation (insurance).
GC — General Contractor.
BMP — Best Management Practice. Erosion/pollution controls during construction.
SWPPP — Stormwater Pollution Prevention Plan.
ASTM — American Society for Testing and Materials. Standards body for construction materials.
D1557 — ASTM test method for soil compaction (Modified Proctor). 90% D1557 is the compaction standard.
SDR-35 — Standard Dimension Ratio pipe rating. Used for subdrain pipe specification.
pcf — Pounds per Cubic Foot. Unit for soil/earth pressure.
psf — Pounds per Square Foot. Unit for soil bearing capacity and wall pressure.
psi — Pounds per Square Inch. Unit for concrete compressive strength (f'c).
f'c — Specified compressive strength of concrete. Minimum 2,500 psi on this project.
ohm-cm — Ohm-centimeters. Unit for soil resistivity/corrosivity. Lower = more corrosive.

Tool Features

Save / Load — Stores all data in your browser's local storage. Persists between sessions as long as you use the same browser.
Export Snapshot — Downloads a standalone HTML file with all current data baked in. Can be shared, archived, or opened on any computer with a browser. No internet required.
Reset to Defaults — Wipes all edits and returns to the original pre-loaded values. Cannot be undone.
Print / PDF — Opens your browser's print dialog. Select "Save as PDF" to create a static document. Print styling hides edit controls and interactive elements.
Custom Columns — Add your own columns to the bid table via the Column Manager below the table. Types: Text, Number, Currency ($), or Formula (auto-calculated from other values).
Add / Delete Rows — Use "+ Add Row" to insert new line items in any section. Use the ✕ button to remove rows. Cannot delete the last item in a section.
Color Coding — Blue headers/tint = Material. Green = Labor. Amber/brown = Equipment. Purple = Duration/Resource. Amber-pale inputs = editable cost fields.
